<!-- DEVELOPMENT -->
## Development

To set up the development environment:

```bash
git clone https://github.com/nebulae-official/betelgeuse.git
cd betelgeuse
pip install uv
uv pip install -r requirements.txt -r dev-requirements.txt
```

### Running Tests

```bash
make test
```

### Linting

```bash
make lint
```

### Building

```bash
make build
```
